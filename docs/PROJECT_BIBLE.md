## Первый уровень — Жилой сектор

Название:
Пробуждение

Цель игрока:
Включить резервное питание станции.

Продолжительность:
5–10 минут.

Основные события:

1. Игрок просыпается в каюте.
2. Исследует коридор.
3. Находит лабораторию.
4. Прослушивает первый аудиодневник.
5. Видит Слепца за стеклом.
6. Получает новую цель — добраться до архива.
