## v0.1.2 — Рефакторинг архитектуры

### Новая структура проекта

Созданы папки:

* src/audio
* src/core
* src/entities
* src/levels
* src/systems
* src/ui

### Система взаимодействий

Создан модуль:

src/systems/interactions.js

Вынесены функции:

* addInteractable()
* removeInteractable()
* getClosestInteractable()

Игровая логика переведена на использование модуля.

Создан файл:

src/core/config.js

Перенесены игровые константы:

* PLAYER_HEIGHT
* CROUCH_HEIGHT
* PLAYER_RADIUS
* GRAVITY
* JUMP_FORCE
* BATTERY_DRAIN_RATE
* BATTERY_PICKUP_CHARGE
* BATTERY_PICKUP_DISTANCE
* FLASHLIGHT_FULL_INTENSITY
* FLASHLIGHT_LOW_THRESHOLD

Стало:

После исправления игра снова запускается корректно.
